# Blue Team Labs

This repository contains my investigation writeups from various **blue team cybersecurity labs** and incident response challenges.

Each folder represents a separate lab where I analyze artifacts such as network traffic, malware samples, or forensic evidence to identify attacker activity.

## Platforms

- CyberDefenders  
- Blue Team Labs Online (BTLO)

## Skills Practiced

- Network traffic (PCAP) analysis  
- Malware and threat intelligence investigation  
- Incident response methodology  
- Basic digital forensics

## Repository Structure

Each lab has its own folder containing the investigation writeup and any supporting screenshots.

Example structure:

```
blue-team-labs
│
├── <lab-name>
│   ├── README.md
│   └── screenshots/
│
├── <lab_name>
│   ├── README.md
│   └── screenshots/
```

New labs will be added as I continue practicing and improving my **blue team investigation skills**.
